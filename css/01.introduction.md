# ការណែនាំអំពី CSS

CSS មានន័យថា **Cascading Style Sheets**។ វាប្រើសម្រាប់រចនារូបរាងរបស់គេហទំព័រ ដូចជា ពណ៌ ទំហំអក្សរ ចន្លោះ layout និងការតម្រឹមមាតិកា។

HTML បង្កើតរចនាសម្ព័ន្ធ។ CSS ធ្វើឱ្យរចនាសម្ព័ន្ធនោះមានរូបរាងស្អាត។

## CSS ប្រើសម្រាប់អ្វី?

CSS អាចប្រើសម្រាប់៖

- ផ្លាស់ប្តូរពណ៌អក្សរ
- ផ្លាស់ប្តូរពណ៌ផ្ទៃខាងក្រោយ
- កំណត់ទំហំ និងប្រភេទអក្សរ
- កំណត់ចន្លោះរវាង element
- រៀបចំ layout
- ធ្វើឱ្យគេហទំព័រមើលទៅស្អាតលើទូរស័ព្ទ និងកុំព្យូទ័រ

## ឧទាហរណ៍ CSS

```css
h1 {
  color: blue;
  text-align: center;
}

p {
  font-size: 18px;
  color: gray;
}
```

## របៀបភ្ជាប់ CSS ជាមួយ HTML

យើងអាចភ្ជាប់ CSS file ទៅ HTML ដោយប្រើ tag `<link>` នៅក្នុង `<head>`។

```html
<head>
  <link rel="stylesheet" href="style.css">
</head>
```

## ចំណុចសំខាន់

- CSS ប្រើសម្រាប់រចនាគេហទំព័រ។
- HTML ជារចនាសម្ព័ន្ធ ចំណែក CSS ជារូបរាង។
- CSS អាចសរសេរនៅក្នុង HTML ឬក្នុង file ដាច់ដោយឡែក។
- ការប្រើ CSS file ដាច់ដោយឡែកធ្វើឱ្យ code ងាយរៀបចំ និងថែទាំ។
