# HTML Elements

HTML element គឺជាផ្នែកមួយនៃគេហទំព័រ។ Element ជាទូទៅមាន opening tag, content និង closing tag។

## រូបមន្តរបស់ Element

```html
<tagname>Content goes here</tagname>
```

ឧទាហរណ៍៖

```html
<h1>Welcome</h1>
<p>This is a paragraph.</p>
```

## ប្រភេទ Element

### Element មាន opening និង closing tag

```html
<p>Hello world</p>
```

### Empty Element

Empty element គឺ element ដែលមិនមាន closing tag។

```html
<br>
<img src="photo.jpg" alt="My photo">
```

## Nested Elements

Nested element មានន័យថា element មួយស្ថិតនៅខាងក្នុង element មួយទៀត។

```html
<div>
  <h2>About Me</h2>
  <p>I am learning HTML.</p>
</div>
```

## ចំណុចសំខាន់

- Element គឺជាផ្នែកនៃ HTML page។
- Element ភាគច្រើនមាន opening tag និង closing tag។
- Element អាចដាក់នៅខាងក្នុង element ផ្សេងទៀតបាន។
- Empty element មិនត្រូវការ closing tag ទេ។
