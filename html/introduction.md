# бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһё HTML

HTML бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ **HyperText Markup Language**бҹ” бһңбһ¶бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”

HTML бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҒбҹ” бһңбһ¶бһҮбһ¶ **markup language** бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ tag бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһ¶бһ”бҹӢ browser бһҗбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ“бһёбһҳбһҪбһҷбҹ—бһ“бҹ…бһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһӮбһәбһҮбһ¶бһўбҹ’бһңбһёбҹ”

## HTML бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһңбһё?

HTML бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбһјбһ…бһҮбһ¶бҹ–

- бһ…бҹҶбһҺбһ„бһҮбһҫбһ„
- бһўбһҸбҹ’бһҗбһ”бһ‘
- бһҸбҹҶбһҺбһ—бҹ’бһҮбһ¶бһ”бҹӢ
- бһҡбһјбһ”бһ—бһ¶бһ–
- бһ”бһүбҹ’бһҮбһё
- бһҸбһ¶бһҡбһ¶бһ„
- Form бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ HTML бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, HTML!</h1>
    <p>This is my first web page.</p>
  </body>
</html>
```

## бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢ

- `<!DOCTYPE html>` бһ”бҹ’бһҡбһ¶бһ”бҹӢ browser бһҗбһ¶бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ HTML5бҹ”
- `<html>` бһҮбһ¶ element бһ’бҹҶбһҮбһ¶бһ„бһӮбҹҒбһҡбһ”бһҹбҹӢбһ‘бҹҶбһ–бҹҗбһҡбҹ”
- `<head>` бһ•бҹ’бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһ‘бҹҶбһ–бҹҗбһҡбҹ”
- `<title>` бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ“бҹ…бһӣбһҫ browser tabбҹ”
- `<body>` бһ•бҹ’бһ‘бһ»бһҖбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһҫбһӣбһғбһҫбһүбҹ”
- `<h1>` бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ’бҹҶбҹ”
- `<p>` бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҗбһ¶бһҒбһҺбҹ’бһҢбҹ”

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢ

- HTML бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”
- HTML бһ”бҹ’бһҡбһҫ tag бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”
- бһҜбһҖбһҹбһ¶бһҡ HTML бһҮбһ¶бһ‘бһјбһ‘бҹ…бһҳбһ¶бһ“ extension `.html`бҹ”
- Browser бһўбһ¶бһ“ HTML бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһҮбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ”
