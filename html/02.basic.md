# មូលដ្ឋានគ្រឹះ HTML

នៅក្នុងមេរៀននេះ យើងនឹងរៀនពីរចនាសម្ព័ន្ធមូលដ្ឋានរបស់ឯកសារ HTML។ គេហទំព័រ HTML មួយត្រូវមាន tag សំខាន់ៗ ដើម្បីឱ្យ browser អាចយល់ និងបង្ហាញមាតិកាបានត្រឹមត្រូវ។

## រចនាសម្ព័ន្ធ HTML មូលដ្ឋាន

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My Heading</h1>
    <p>My paragraph.</p>
  </body>
</html>
```

## Tag សំខាន់ៗ

- `<!DOCTYPE html>` បញ្ជាក់ថាឯកសារនេះជា HTML5។
- `<html>` គ្របដណ្តប់មាតិកា HTML ទាំងមូល។
- `<head>` ផ្ទុកព័ត៌មានដែលមិនបង្ហាញផ្ទាល់លើទំព័រ។
- `<title>` បង្ហាញចំណងជើងនៅលើ browser tab។
- `<body>` ផ្ទុកមាតិកាដែលបង្ហាញលើទំព័រ។

## HTML Tag

Tag ភាគច្រើនមាន opening tag និង closing tag។

```html
<p>This is a paragraph.</p>
```

ក្នុងឧទាហរណ៍នេះ៖

- `<p>` គឺ opening tag។
- `This is a paragraph.` គឺមាតិកា។
- `</p>` គឺ closing tag។

## ចំណុចសំខាន់

- HTML page ត្រូវមានរចនាសម្ព័ន្ធច្បាស់។
- មាតិកាដែលមើលឃើញត្រូវដាក់ក្នុង `<body>`.
- Tag ជួយប្រាប់ browser ថាមាតិកានោះជាប្រភេទអ្វី។
