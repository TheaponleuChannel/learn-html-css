# HTML Style

HTML style attribute ប្រើសម្រាប់បន្ថែមការរចនាទៅលើ element ដោយផ្ទាល់។ Style ក្នុង HTML ប្រើ syntax របស់ CSS។

## របៀបសរសេរ Style

```html
<tagname style="property: value;">Content</tagname>
```

ឧទាហរណ៍៖

```html
<p style="color: red;">This text is red.</p>
```

## កំណត់ពណ៌អក្សរ

```html
<h1 style="color: blue;">Blue Heading</h1>
```

## កំណត់ពណ៌ផ្ទៃខាងក្រោយ

```html
<p style="background-color: yellow;">Yellow background</p>
```

## កំណត់ទំហំអក្សរ

```html
<p style="font-size: 20px;">Large text</p>
```

## កំណត់ការតម្រឹមអត្ថបទ

```html
<h1 style="text-align: center;">Centered Heading</h1>
```

## ចំណាំ

ការប្រើ `style` attribute គឺងាយស្រួលសម្រាប់ការរៀនដំបូង ប៉ុន្តែពេលបង្កើតគេហទំព័រធំៗ យើងគួរប្រើ CSS file ដាច់ដោយឡែក។

## ចំណុចសំខាន់

- `style` attribute ប្រើសម្រាប់រចនា element មួយ។
- Style ប្រើទម្រង់ `property: value;`។
- CSS file ដាច់ដោយឡែកល្អជាង inline style សម្រាប់ project ធំ។
