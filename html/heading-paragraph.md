# HTML Headings and Paragraphs

Headings бһ“бһ·бһ„ paragraphs бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҳбһ¶бһҸбһ·бһҖбһ¶бһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбҹ” Heading бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ„бһҮбһҫбһ„ бһ…бҹҶбһҺбҹӮбһҖ paragraph бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ”

## HTML Headings

HTML бһҳбһ¶бһ“ heading бһ–бһё `<h1>` бһҠбһӣбҹӢ `<h6>`бҹ”

```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Small Heading</h3>
```

`<h1>` бһҮбһ¶бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ’бҹҶбһ”бҹҶбһ•бһ»бһҸ бһ бһҫбһҷ `<h6>` бһҮбһ¶бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһҸбһјбһ…бһ”бҹҶбһ•бһ»бһҸбҹ”

## HTML Paragraphs

Paragraph бһ”бҹ’бһҡбһҫ tag `<p>`бҹ”

```html
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
```

Browser бһ“бһ№бһ„бһҠбһ¶бһҖбҹӢбһ…бһ“бҹ’бһӣбҹ„бҹҮбһҡбһңбһ¶бһ„ paragraph бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бҹҒбһү

```html
<h1>About HTML</h1>
<p>HTML is used to create web page structure.</p>

<h2>Why Learn HTML?</h2>
<p>HTML is the first step in web development.</p>
```

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢ

- бһ”бҹ’бһҡбһҫ heading бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бҹ”
- бһ”бҹ’бһҡбһҫ `<h1>` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”
- бһ”бҹ’бһҡбһҫ `<p>` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҗбһ¶бһҒбһҺбҹ’бһҢбҹ”
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ heading бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҳбһ¶бһҸбһ·бһҖбһ¶бһ„бһ¶бһҷбһўбһ¶бһ“бҹ”
