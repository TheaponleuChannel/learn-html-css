## លំដាប់នៃការរៀន

1. ចាប់ផ្តើមពី HTML ដើម្បីយល់ពីរចនាសម្ព័ន្ធរបស់គេហទំព័រ។
2. រៀនពី element, attribute, heading, paragraph, formatting និង style ក្នុង HTML។
3. បន្តទៅ CSS ដើម្បីរៀនពីការរចនាពណ៌ ទំហំ អក្សរ និង layout។
4. បន្ទាប់ពីរៀនមេរៀននីមួយៗ គួរអនុវត្តដោយបង្កើតឧទាហរណ៍តូចៗ។
